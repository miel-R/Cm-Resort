The CM Resort
> ⚠️ Late Post Notice > > This project was originally hosted on my educational GitHub account which is now disabled. I'm sharing it here to keep a record of my work and make it publicly accessible.

## Features
- Browse available rooms and amenities
- Make reservations for specific dates
- Manage existing bookings
- Track guest information
- User-friendly web interface

## Tech Stack
- Backend: PHP
- Frontend: HTML, CSS, JavaScript
- Server: Apache / XAMPP / LAMP stack
- Database: MySQL (or MariaDB)
